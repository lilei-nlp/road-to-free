# 我的投资实践

> 以下内容仅为个人实践记录，不构成任何投资建议。每种策略都有其适用的市场环境和风险特征。

## 一、资产配置总体思路

我的资产配置遵循一个四层结构——这个框架在第 3 章已经描述过，这里放到实践里再展开：

```
总资产
├── 安全垫（现金/短期理财）      ~20%
│   6-12 个月生活费，绝不动用
├── 稳健仓（美股指数基金）        ~40%
│   定投，长期持有，不择时
├── 增强仓（可转债轮动）          ~25%
│   主动管理，追求超额收益
└── 探索仓（港股打新/个股）       ~15%
   小仓位，学习与发现机会
```

**核心原则：永远先确定安全垫，再考虑收益。**

很多人犯的错误是一上来就琢磨"买什么赚钱"，却忽略了最重要的事——万一亏了，你扛不扛得住？安全垫决定了你在市场最差的时候会不会被迫卖出（而被迫卖出往往是永久的亏损）。

## 二、可转债轮动实践

### 什么是可转债？

可转债，全称"可转换公司债券"，是上市公司发行的一种特殊债券。它有两个身份：

1. **债**——发行人到期还本付息，持有到期至少能拿回 100 元面值 + 利息
2. **股**——持有人有权按约定价格（转股价）把债券换成股票。股票涨了，你也跟着赚

简单说：**跌了有人兜底，涨了你跟着吃肉。** 这种收益结构在金融产品里极为罕见——绝大多数产品都是"高风险高收益、低风险低收益"，可转债是唯一一个**风险收益不对称**的品种。

### 下有保底，上不封顶

这是可转债最核心的魅力。

- **向下**：只要发行公司不违约，转债持有到期能拿回本息（目前中国可转债历史上零违约）
- **向上**：正股大涨，转债可以转股兑现，理论上没有涨幅上限

想象一下：一个品种跌最多跌 10-15%（到债底），但涨能涨 50%、100% 甚至更多。这就是**非对称收益**——胜率和盈亏比天然站在你这边。

### 下修：可转债最大的"Bug"

可转债有一个大多数散户不知道的条款：**下修转股价**。

当正股价格跌得很惨，转债的转股价值变低时，发行人可以选择**向下修正转股价**。一旦下修，每张转债能换更多股票，转债价格立刻跳涨。

发行人为什么愿意下修？因为他们不想还钱。可转债到期是要还本的，上市公司从借到钱的那一天起，就巴不得你全部转股——转股之后债务清零，不用还了。

**统计规律来看，A 股历史上绝大多数可转债最终以"强赎转股"收场——发行人故意让股价超过转股价的 130%，逼你转股。** 借钱不想还，这是一种人性，也是一种规律。

### 我的轮动策略

最经典的是**双低策略**：低价格 + 低溢价率。

- **低价格**：债底附近买入，下行空间锁死
- **低溢价率**：转股溢价率越低，转债跟涨正股越紧，弹性越好

定期轮动，每月或每季度调仓一次：卖出评分下降的，补入新符合条件的。不用盯盘，不用预测大盘涨跌，策略规则化执行即可。

目前市场越来越成熟，简单的双低策略收益在收窄，需要更复杂的多因子模型（加入正股动量、波动率、剩余规模等因子）。但核心逻辑没变——可转债的非对称收益结构决定了它相比 A 股正股，是一个**更优质的底层资产**。

### 为什么可转债值得 PhD 深入研究

相比 A 股散户的生存环境（信息劣势、波动剧烈、策略难持续），可转债有几个结构性优势：

1. **规则透明**——条款白纸黑字，不需要预测涨跌，只需要算清楚债底、溢价率、到期收益率
2. **容错率高**——错了亏有限（有债底），对了赚得多（强赎触发），反复做期望值为正
3. **适合量化**——因子可以回测，策略可以迭代，天然适合 PhD 的思维和工具能力
4. **竞争相对温和**——相比 A 股正股，可转债市场的参与者结构对散户相对友好

## 三、美股优质股票 + Covered Call 实践

### 核心：QQQ / VOO 作为底仓

美股仓位的主要部分始终是指数 ETF 定投：**Nasdaq 100 (QQQ)** 和 **S&P 500 (VOO)**。每月固定金额，不择时。

对于 PhD 来说，你最好的定投时间是你做实验、写论文的那几年——市场涨跌跟你无关，你只是在每个月默默买入。

### 利用 TQQQ 加速回撤修复

很多人对杠杆 ETF 有莫名的恐惧，觉得杠杆 = 爆仓 = 危险。但 TQQQ 跟期货期权不同——它是 3 倍做多 Nasdaq 100 的 ETF，**持有它不会爆仓、不会被追加保证金**。你买入一股 TQQQ，最坏的情况是跌得很惨，但它不会归零，也不会强迫你平仓。

这意味着你可以把它当成一个**战术工具**，在特定场景下使用：

**策略：大跌时部分切换，反弹修复后换回。**

假设你有 ¥100,000 的 QQQ，Nasdaq 100 下跌了 20%：

```
QQQ 从 $400 跌到 $320（-20%）
你的持仓：¥100,000 → ¥80,000（浮亏 ¥20,000）
```

此时你把 25% 的仓位从 QQQ 切换到 TQQQ：

```
¥80,000 持仓中：
    75% QQQ：  ¥60,000（150 股 × $320）
    25% TQQQ： ¥20,000
```

然后 Nasdaq 100 从 $320 反弹回 $400（+25%）：

```
QQQ：  +25%     → 150 股 × $400 = $60,000
TQQQ：+75%（3x）→ ¥20,000 × 1.75 = ¥35,000
总市值：¥95,000
```

如果没有切换，纯 QQQ 回到 ¥100,000。切换后看起来少了 ¥5,000？不——**因为你只需要再涨一点点就能回到 ¥100,000，而纯 QQQ 才刚刚回本。**

更重要的是，如果反弹幅度大于 25%（市场超跌反弹往往很猛烈），TQQQ 的超额收益会非常明显。弹回 $400 后把 TQQQ 换回 QQQ，锁定这轮反弹加速修复的部分，然后恢复到底仓模式。

**关键约束（不要滥用）：**

1. **只在显著回撤时使用**——QQQ 跌 20%+ 才考虑。小跌不操作
2. **只用部分仓位**——25% 左右，不是全仓切。留有容错空间
3. **反弹目标达成即换回**——TQQQ 是战术工具，不是长期持仓。长期持有 3x 杠杆 ETF 有衰减损耗
4. **如果继续跌呢？** 再跌 20% 可以再切一部分。但永远不要全仓杠杆

> 这个策略的核心不是预测涨跌，而是**利用杠杆加速已有回撤的修复**。美股历史上每一次超过 20% 的回调都会被收复——问题只是时间。TQQQ 帮你缩短这个时间。

### 优质个股 + Covered Call

在指数仓位之外，我会挑选一些**优质美股个股**长期持有——通常是护城河深、现金流好、你真正理解其商业模式的科技公司。

在此基础上，我会卖出这些持仓的 **Covered Call（备兑看涨期权）**：

> 持有正股 → 卖出对应张数的看涨期权 → 收取权利金

简单理解：你承诺在未来某一天以某个价格卖出你的股票，作为交换，你**立刻收到一笔现金**（权利金）。如果到期股价没涨到那个价格，期权作废，你白赚权利金；如果涨到了，你按约定价卖出，赚了权利金 + 股价差价。

**Covered Call 的本质：在长期持有优质股票的基础上，把"闲置的波动性"变现。**

你不预测涨跌，你只是在已经持有股票的前提下，顺便收租。适合波动不大、长期向上的优质标的。风险是卖飞——股票暴涨超出行权价，你少赚了。但如果你本来就想在那个价位卖出，这就不是风险。

### 这块我还在持续学习中

期权是一个深水区。Covered Call 是最温和的起点——它不需要保证金，不会爆仓，风险可控。但即使如此，也需要充分理解到期日、行权价、隐含波动率这些概念之后再动手。

## 四、港股打新

### 背景：港股打新的回暖

港股 IPO 打新在 2019-2021 年经历了一轮黄金期，随后 2022-2024 年市场降温，优质标的减少。但从 **2025 年中开始**，随着港股市场情绪回暖、一批优质公司重启 IPO，打新套利的机会重新出现了。

### 港股打新规则简析

港股 IPO 的配售机制对散户有两组分配：

| | 甲组 | 乙组 |
|------|------|------|
| 申购金额 | 500 万港币以下 | 500 万港币以上 |
| 中签率 | 较低，抽签分配 | 较高，按比例分配 |
| 适合人群 | 小资金试水 | 大资金提高确定性 |

港股打新采用的是**混合分配机制（Mechanism B）**——在回拨触发后，公开发售部分会从国际配售"回拨"一定比例给散户（通常是 10%→30%→50%），散户分配池变大，中签率提升。当超额认购倍数极高时，回拨至 50% 意味着散户拿走了半只新股的筹码。

### 我的打新策略：选择性参与 + 乙组提高概率

不是每只新股都打，选择标准：

- **基本面扎实**：公司有清晰盈利模式，不是概念炒作
- **估值合理**：发行价相比可比公司有折价空间
- **有基石投资者**：有机构背书，意味着上市后有买盘支撑
- **市场情绪好**：大盘不在恐慌下跌期

一旦确认要打，我会**通过融资进入乙组**（单次申购金额超过 500 万港币）。乙组的优势是中签率高且按比例分配——优质的标的你值得拿更多，鸡肋的标的你根本不应该参与。

> 打新的核心原则：**宁缺毋滥。** 大多数新股不值得打。一年真正值得参与的优质 IPO 可能只有 3-5 只。在那些时候重仓参与，比每只都打效果好得多。

## 踩过的坑

1. **追涨杀跌**——刚开始投资时，看到涨就买，看到跌就卖。学费交了大概 10%
2. **过度交易**——频繁调仓，交易成本吃掉了一部分收益
3. **信息焦虑**——每天刷财经新闻，其实 99% 的信息对长期策略无用
4. **过早卖出盈利标的**——行为金融学里叫"处置效应"，盈利了急着落袋，亏损了舍不得卖
