# 理解周期

第 3 章提到：投资就是找到一揽子低相关性的资产，在周期的起伏中低买高卖。资产配置是"拿什么"，周期是"什么时候拿"——这一章专门讲周期。

## 为什么要理解周期？

因为**大多数投资错误不是在选择品种上犯的，而是在错误的时间做了正确的事。**

2007 年 6000 点满仓 A 股、2021 年追高核心资产——回头看，买的都是好资产，但买在了周期的顶点。反过来，2018 年底、2020 年 3 月、2022 年 10 月——遍地便宜筹码，但大多数人因为恐慌不敢下手。

> **周期不会告诉你"明天涨还是跌"，但它会告诉你"我们现在大概在什么位置"。**

## 霍华德·马克斯的周期框架

《周期》这本书给出了理解市场节奏的最清晰框架：

### 两个核心钟摆

1. **情绪的钟摆**——在"贪婪"和"恐惧"之间来回摆动。极少停在中间的"理性"位置
2. **风险态度的钟摆**——在"无视风险"和"极度厌恶风险"之间摆动

市场顶部时，人们说"这次不一样，估值不重要"；市场底部时，人们说"这次是真的完了"。这两句话每一次都会出现，而每一次最终都被证明是错的。

### 判断位置的方法

不需要精确的时间，只需要模糊的位置感：

| 信号 | 位置判断 |
|------|------|
| 大家都在讨论"这次不一样" | 大概率接近顶部 |
| 媒体标题全是恐慌 | 大概率接近底部 |
| IPO 破发潮、新基金卖不动 | 寒冬，性价比高的时候 |
| 同事开始给你推荐股票 | 过热信号 |
| 有人说"现金是垃圾" | 极度贪婪 |

> 这些信号单独看任何一个都不构成买卖依据，但**多个信号叠加在一起**，周期位置就很清晰。

## 不同类型的周期叠加

真正值得认真对待的是，不同资产有不同的周期，它们有时**共振**，有时**错位**：

### A 股的周期

A 股最剧烈的变量是**政策周期**：宽松周期（放水）→ 资产价格涨 → 监管收紧 → 去杠杆 → 跌 → 重新宽松。这个节奏 5-7 年一轮，比美股的商业周期短得多，振幅大得多。

### 美股的周期

美股主要由**盈利周期**和**联储利率周期**驱动。历史上牛长熊短，但熊市来的时候往往很猛烈且短暂（2020 年 3 月的暴跌只持续了一个月）。

### 可转债的周期

可转债有自己独立的供需周期：
- **供给端**：什么时候上市公司密集发转债
- **需求端**：机构配置资金的进出
- **信用周期**：转债价格受信用风险定价变化的影响

### 为什么"几类资产低相关"那么重要？

如果所有资产都在同一个周期（比如 2020 年的全球大放水，股债商品一起涨），那分散配置就失效了。

但如果 A 股在挤泡沫而美股在 AI 驱动上涨、可转债因为信用修复正在修复折价——三个品种的时间线错开了，你就可以在周期的不同阶段**把资金从贵的资产搬到便宜的资产上**。

**真正的超额收益不是选对一只股票，而是在品种之间、在周期之间做切换。**

## 如何利用周期但不迷信周期？

1. **知道大概位置就够**——不需要择时到精确的月，模糊的正确优于精确的错误
2. **极端位置才做大动作**——普通的估值中枢波动，不需要操作；只在极度便宜的时候加仓、极度昂贵的时候减仓
3. **永远不去预测底部和顶部**——"买在最低、卖在最高"的想法是亏损的源头
4. **周期可以让你多赚，但安全垫是你活下去的前提**——你在寒冬里逢低加仓的前提，是你不会因为恐慌或生活压力被迫卖出
