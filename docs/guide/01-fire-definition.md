# FIRE 的数学定义

## 什么是 FIRE？

FIRE 是 Financial Independence, Retire Early 的缩写——财务独立，提早退休。

但它的核心不是"不工作"，而是**拥有选择权**：你可以继续做你喜欢的研究，但你不再需要为钱做任何你不想做的事。

## 4% 法则

FIRE 的理论基础是 Trinity Study 提出的 4% 安全提款率（Safe Withdrawal Rate）：

> 如果你的年支出为 $X，那么你需要的净资产 = X × 25。每年提取 4%，本金大概率不会耗尽。

这是 FIRE 最核心的数学公式。一切策略都围绕这个公式展开。

## 你的数字：为什么 100 万美金？

假设一个理性且舒适的年度开销：

| 类别 | 月支出（USD） | 年支出（USD） |
|------|:---------:|:---------:|
| 房租/房贷 | 1,200 | 14,400 |
| 饮食 | 600 | 7,200 |
| 交通 | 200 | 2,400 |
| 医疗/保险 | 400 | 4,800 |
| 娱乐/旅行 | 500 | 6,000 |
| 其他 | 100 | 1,200 |
| **合计** | **3,000** | **36,000** |

$36,000 × 25 = **$900,000**。留一点安全边际，100 万美金是一个合理且可实现的目标。

> 当然，这里的支出模型因人而异。如果你生活在低消费城市或无家庭负担，这个数字可以更低。关键不是具体数字，而是**建立"目标资产 = 年支出 × 25"这个思维框架**。

## PhD 的天然优势

作为 PhD，你在 FIRE 这件事上有几个几乎不公平的优势：

1. **已经习惯了低成本生活**——不需要戒奢侈消费，因为你本来就没有
2. **收入增长曲线陡峭**——从 stipend 到 intern 到 full-time，收入翻几倍只需几年
3. **理性决策能力**——你受过系统性的科学训练，不容易被情绪和市场噪音左右

**FIRE 不是关于"赚多少钱"，而是关于"支出与资产的比值"。** PhD 在这个比值上，起点已经比别人高了一大截。

## 路线图

```
年支出 $X → 目标资产 X×25 → 年储蓄率 → 预估年限
```

简单估算：假设你年储蓄 $100,000，目标 $1,000,000，年化收益 7%，大约需要 **7-8 年**。

如果你能通过投资把年化收益提升到 10-12%，或者通过副业提高储蓄率，这个时间还能缩短。
