# 培养投资范式

## 为什么大多数 PhD 空有收入却没投资体系？

这是个很普遍的现象。原因很简单：

1. **信息过载**——投资领域的信息噪声比学术论文大多了，不知道从哪开始
2. **过度分析**——PhD 通病，想把所有变量都建模，结果永远在"研究"阶段
3. **怕犯错**——学术训练让我们对"错误"高度敏感，但投资中犯错是必修课

## 我的投资学习路径

### 第一阶段：阅读经典（1-3个月）

不要看财经新闻，不要听 Youtuber 推荐个股。先读经典：

- 《漫步华尔街》— 有效市场假设，为什么大多数主动投资跑不赢指数
- 《聪明的投资者》— 价值投资的基础框架
- 《投资最重要的事》— 风险、周期、心理
- 《共同基金常识》— 低成本指数投资的逻辑

> 读完这些你会意识到：**投资不需要你多聪明，需要你够理性。**

### 第二阶段：模拟 + 小资金（3-6个月）

- 开一个模拟账户，练习资产配置决策
- 用一笔"全亏了也不心疼"的小钱开始真实交易
- 这个阶段的目标不是赚钱，是**建立肌肉记忆**：下单、止损、持仓过夜、经历波动

### 第三阶段：形成框架（持续迭代）

在有了基本的市场感知后，开始建立你自己的投资决策框架。我的框架包括：

## 资产配置框架

```
总资产
├── 安全垫（现金/短期国债）
│   └── 6-12 个月生活费，绝不动用
├── 稳健仓（指数基金/债券）
│   └── 定投，长期持有，不择时
├── 增强仓（可转债/Reits/红利策略）
│   └── 轮动，有策略地主动管理
└── 探索仓（个股/打新/套利）
    └── 小仓位，用于学习和发现机会
```

这个框架的好处是：
- **安全垫**让你在市场最差的时候也不会被迫卖出
- **稳健仓**保证基本收益，让你睡得着觉
- **增强仓**提供超额收益的可能
- **探索仓**满足学习需求，亏损有上限

## 风险管理

比赚钱更重要的，是不亏大钱。几条铁律：

1. **永远不要上杠杆**——尤其不要用期权、期货放大风险
2. **单只标的不超过总资产的 10%**——分散是唯一免费的午餐
3. **有止损计划**——买入前就想好什么情况下卖出
4. **不要把应急资金投入市场**——安全垫是不可触碰的底线

## 复利的力量

```
100,000 × (1.10)^20 = 672,750
100,000 × (1.07)^20 = 386,968
```

同样是 10 万本金，10% 年化 vs 7% 年化，20 年后差了近一倍。但——

**比收益率更重要的是本金规模。** 早期多攒 10 万，远比把收益率从 7% 提高到 10% 更实际。先积累，再优化。
