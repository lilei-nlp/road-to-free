# 推荐书单

> 使用建议：不需要逐本读完。先扫一遍，标记你当前最关心的 2-3 本，读完再说。

| 书名 | 作者 | 一句话 |
|------|------|--------|
| 《小狗钱钱》 | Bodo Schäfer | 建立对钱的最基础认知：储蓄、分配收入、用钱生钱 |
| 《穷爸爸富爸爸》 | Robert Kiyosaki | 理解"资产"和"负债"的区别——这个区别会改变你对一切支出的看法 |
| 《漫步华尔街》 | Burton Malkiel | 最好的投资入门书，读完会理解为什么指数投资是大多数人的最优解 |
| 《聪明的投资者》 | Benjamin Graham | 价值投资圣经。"安全边际"和"市场先生"两个概念受用终身 |
| 《投资最重要的事》 | Howard Marks | 关于风险、周期和心理的洞见，几乎每一页都值得划线 |
| 《共同基金常识》 | John Bogle | 低成本、长期持有、指数化——朴素但极有效 |
| 《周期》 | Howard Marks | 理解市场情绪钟摆和风险态度周期，知道自己在什么位置比知道明天涨跌重要 |
| 《非理性繁荣》 | Robert Shiller | 对泡沫和市场情绪的经典分析 |
| 《随机漫步的傻瓜》 | Nassim Taleb | 很多成功只是运气，别错把风口当成自己的能力 |
| 《黑天鹅》 | Nassim Taleb | 极端事件比你以为的更常见，你的体系必须能扛住它 |
| 《反脆弱》 | Nassim Taleb | 不只是扛住风险，而是从混乱和波动中受益 |
| 《思考，快与慢》 | Daniel Kahneman | 你犯过的每一个投资错误，基本都能在这本书里找到原因 |
| 《穷查理宝典》 | Charlie Munger | 多元思维、逆向思维、人类误判心理学 |
| 《金钱心理学》 | Morgan Housel | 财富不是你赚了多少，而是你留下了多少 |
| 《纳瓦尔宝典》 | Naval Ravikant | 关于财富、杠杆和自由。三小时翻完，值得反复看 |
| 《Your Money or Your Life》 | Vicki Robin | FIRE 运动奠基作。你花掉的不是钱，是赚钱的生命时间 |
| 《The Simple Path to Wealth》 | JL Collins | 最简单的 FIRE 方案：存钱 → 买指数 → 等着 |
| 《Die With Zero》 | Bill Perkins | FIRE 的反面：不要为了攒钱错过人生体验 |
| 《可转债投资魔法书》 | 安道全 | 可转债入门，讲清"下有保底、上不封顶"的底层逻辑 |
| 《攻守》 | 饕餮海 / 定风波 / 优美 | 进阶可转债策略，适合想深入轮动模型的读者 |
| 《快学期权》 | — | 了解期权基本概念，不一定需要参与 |
| 《投资者的未来》（中文版） | 唐朝 | 适配 A 股/港股市场的价值投资框架 |
| 《您厉害，您赚得多》 | 不明真相的群众 | 雪球创始人碎片式思考，轻松但有启发 |

## 在线资源

|------|------|--------|
| 《投资第一课》 | 有知有行 | 国内最好的入门课程。系统、克制、没有荐股，学完可做题自检 |
